# Pension & Gratuity Calculation Engine

Automated financial calculation engine built for Defined Benefit (DB) Tech domain compliance. This tool calculates pension and gratuity based on standardized financial formulas, includes an automated QA validation suite, and features a user-friendly interface.

## 🚀 Key Features
- **Core Engine:** Python-based logic with 100% accuracy on eligibility rules.
- **Automated Testing:** Integrated `unittest` framework to validate all financial business rules.
- **Batch Processing:** Handles CSV uploads to calculate benefits for thousands of employees in seconds.
- **User Dashboard:** Interactive UI built with Streamlit for real-time validation.

## 🛠 Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, Streamlit, Unittest
- **Deployment:** Streamlit Community Cloud

## 📈 How to run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the UI: `streamlit run app.py`
